# Tutorial de Programación en Java

## Descripción

Este proyecto fue desarrollado como material educativo para reforzar conocimientos básicos de programación en Java.

El contenido está dirigido a estudiantes que están comenzando a programar y necesitan ejemplos sencillos para comprender conceptos fundamentales.

## Objetivo

Crear una guía de consulta sobre temas básicos de Java mediante explicaciones teóricas y ejemplos prácticos.

## Temas Desarrollados

### 1. Ciclos (Bucles)

Los ciclos permiten repetir instrucciones múltiples veces.

Tipos estudiados:

- For
- While
- Do While

Archivos:

- codigo/ciclos/EjemploFor.java
- codigo/ciclos/EjemploWhile.java
- codigo/ciclos/EjemploDoWhile.java

---

### 2. Métodos (Funciones)

Los métodos permiten organizar el código y reutilizar instrucciones.

Archivos:

- codigo/metodos/MetodoSaludar.java
- codigo/metodos/MetodoSuma.java
- codigo/metodos/MetodoRetorno.java

---

### 3. Vectores y Matrices

Permiten almacenar múltiples valores dentro de una misma estructura.

Archivos:

- codigo/vectores_matrices/VectorBasico.java
- codigo/vectores_matrices/RecorrerVector.java
- codigo/vectores_matrices/MatrizBasica.java

---

### 4. Listas (ArrayList)

Permiten almacenar elementos de manera dinámica.

Archivos:

- codigo/listas/ListaBasica.java
- codigo/listas/AgregarElementos.java
- codigo/listas/RecorrerLista.java

---

### 5. Manejo de Archivos

Permite leer, escribir y procesar información almacenada en archivos de texto.

Archivos:

- codigo/archivos/LeerArchivo.java
- codigo/archivos/EscribirArchivo.java
- codigo/archivos/ProcesarArchivo.java

---
